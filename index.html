Input Anggota Keluarga

Level 1 menggunakan data statis dengan 3 anggota keluarga utama.

  • Nabi Muhammad
  • Abdullah (Ayah)
  • Aminah binti Wahb (Ibu)
Untuk level lanjutan, input dapat dikembangkan menjadi dinamis menggunakan form dan struktur data yang lebih kompleks.

Panduan Aplikasi (Sample)

  1. Buka menu Input Anggota Keluarga
  2. Perhatikan struktur parent dan child
  3. Lihat visualisasi tree pada menu Visualisasi
  4. Pelajari algoritma dan struktur data yang digunakan

Algoritma

Buat Algoritma utama aplikasi! Pilih bentuk algoritma yang paling kamu suka:

  • Flowchart (gambar JPG)
  • Pseudocode
Contoh pseudocode: 
START
Buat node Ayah
Buat node Ibu
Buat node Anak
Hubungkan Ayah dan Ibu sebagai parent
Hubungkan Anak sebagai child
Tampilkan struktur tree
END
